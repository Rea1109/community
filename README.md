# Community web App

[프로젝트 보러가기](https://rea-community.herokuapp.com/)  
React , Typescript 기반 게시판 프로젝트 (진행중)

## 프로젝트 실행

```javascript

    // backend db mock server
    npm install -g json-server

    npm install
    npm start
```

## 프로젝트 개요

> React , Typescript , react-router-dom 기반 게시판
>
> 기본적인 CRUD 구현 목표
